SQL

SQL (Structured Query Language) is an industry standard language designed specifically to manage the flow of data within databases (Wikimedia Foundation), and you can expect to use it (or in a library for another language!)* as a Data Scientist.
*Examples include;
-> sqlite for C (SQLite)
-> sqlite3 for Python (Python Docs)
You can use programs like PostgresSQL or MySQL to work with... guess what... SQL.

Jupyter

Jupyter is a browser based application that allows developers program snippets of code in various different 'cells' using many different programming languages (mainly Python)
The project is mainly focused towards developers who are not looking towards creating large software projects for a general userbase, but rather looking to make segmented pieces of code that perform specific individual jobs for other engineers/developers.
The reason this project helps Data Scientists specifically is in how peices of code can be segmented into seperate cells; this gives Data Scientists the ability to seperate sets of data and any experiments, tests, simulations, etc.
